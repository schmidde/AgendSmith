AgendSmith for Twitter

AgendSmith.jar wird über die Kommandozeile aufgerufen.
Dazu ins Verzeichnis mit der .jar-Datei wechseln und 

java -jar AgendSmith.jar 

eingeben.

Die Anwendung kann mit 2 Modi benutzt werden. 
Eine Übergabe von Parametern beim Programmaufruf ist nicht nötig.

Modus 1:

Hier wird nach einer ID gefragt.
Als Bsp. kann 123456789 eingegeben werden und es wird die Timeline des User angezeigt.

Modus 2:

Hier werden die Directnachrichten angezeigt.
Wenn eine Nachricht eingegangen ist wird diese beantwortet.

Beenden:

Mit Eingabe von 0 wird das Programm beendet
